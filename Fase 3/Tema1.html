Fundamentos de HTML

HTML 5, son las siglas en inglés para Hyper Text Markup Language o Lenguaje de marcas de texto.
Como menciona Celaya en su libro creación de paginas web, un documento HTML no es mas que
un archivo de texto, por lo que tiene la versatilidad de que se puede trabajar con cualquier editor
de texto. Si bien en la actualidad menciona también que existen herramientas mas visuales no las
recomienda para el aprendizaje, dado que el ejercicio de autocompletar puede afectar la calidad
del aprendizaje de HTML.

Para crear un archivo HTML basta con crear un archivo de texto común (.txt) y modificar su
extensión a .html, por ejemplo libro.txt a libro.html, importante tener presente que ejecutar este
archivo abrirá una pagina web, por lo que para editarlo basta con ir al bloc de notas (o el editor de
texto usado), buscar la opción de abrir y ubicar el elemento .html que queramos editar.

Interaccion Humano-Ordenador

La interacción humano ordenador, es como su nombre indica la disciplina que estudia como las
personas interactúan con las computadoras y como estas ultimas se desarrollan para esta
interacción.

Ya desde el punto del diseño, como menciona Granollers dentro del libro diseño de sistemas
interactivos centrados en el usuario, la fase del diseño es parte de la vida del ciclo del software, es
en esta etapa donde posterior a realizar varias pruebas en base a los requisitos que se define la
usabilidad y funcionalidad, conceptos primordiales para la interacción del humano con el sistema.

Solo teniendo claros los aspectos de usabilidad y funcionalidad es que si diseña el sistema, no al
contrario, la intención, es entonces, que el sistema pueda ser usable de manera sencilla y que
funcione para cumplir los requisitos definidos previo a su creación.
